Ember AI Prompt Assistant App

Ember AI is an AI prompt assistant app built with React TypeScript Tailwind CSS and Next.js and is designed to help a user craft a well written, complete prompt to use when utilizing an AI platform in seeking information. It is based on the Pentagram method, or framework of crafting a prompt, which is composed of five parts: Persona, Context, Task, Output, and Constraints.

Skills: React, TypeScript, Tailwind CSS, Next.js

Tools: VS Code, Gemini API, GitHub, Vercel
